# Hertz & Hearts 1.0.0-beta (Call for Testers)

Hertz & Hearts **1.0.0-beta** is now open for public beta testing.

Built upon OpenHRV (Jan C. Brammer), continued under GPL-3.0.

This is a desktop HRV biofeedback app for research and educational workflows using chest-strap ECG sensors.

**Important:** Research use only. Not for clinical diagnosis or treatment.

## What We Need From Beta Testers

We are validating stability, usability, and export/report workflows across real hardware and operating systems.

Please run through:

- `docs/PUBLIC_RELEASE_CHECKLIST.md`

Then submit:

- bugs via GitHub Issues (`bug_report.yml`), and/or
- beta checklist feedback via GitHub Issues (`beta_checklist_feedback.yml`)

## Good Testers For This Beta

- HRV/biofeedback practitioners
- breathwork and coaching practitioners
- fitness/recovery users who track trends
- developers/testers comfortable reporting reproducible issues

## What To Include In Feedback

- OS version and app version
- sensor model
- pass/fail checklist items
- exact repro steps for any failures
- screenshots or short clips when possible

## Beta Scope

This beta focuses on:

- session flow (connect -> record -> stop/save)
- Session History and replay
- report generation (including report rebuild from past sessions)
- reliability during connect/disconnect/reconnect flows

Not in scope for beta sign-off:

- clinical claims or interpretation
- broad feature expansion

## Where To Start

1. Install latest beta build from Releases.
2. Run the checklist in `docs/PUBLIC_RELEASE_CHECKLIST.md`.
3. Open an issue with findings.

Thank you for helping harden Hertz & Hearts for a stable public release.
