# Show HN Draft (Public Beta)

## Title

Show HN: Hertz & Hearts 1.0.0-beta.1 - Open-source desktop HRV biofeedback app

## Body

Hi HN,

I am releasing **Hertz & Hearts 1.0.0-beta.1**, an open-source desktop app for HRV biofeedback sessions using ECG chest straps (e.g., Polar H10).

This beta is mainly for validating:
- session reliability (connect/record/stop/save)
- history/replay workflow
- report generation and rebuild from saved sessions
- reconnect edge cases

I would appreciate testers willing to run a structured checklist and file actionable issues.

Project:
- https://github.com/JoelAtHome/HertzAndHearts

Tester checklist:
- `docs/PUBLIC_RELEASE_CHECKLIST.md`

Feedback templates:
- bug report
- beta checklist feedback

Important disclaimer:
- Research use only.
- Not for clinical diagnosis or treatment.

If you test it, details like OS version, sensor model, and exact repro steps are incredibly helpful.

Thanks for taking a look.
