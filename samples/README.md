# Samples

These files are safe, synthetic examples for testing import workflows.

## Included files

- `native_sample.csv`: Hertz & Hearts native event/value/timestamp format.
- `rr_only_sample.txt`: Line-separated RR intervals in milliseconds.

## Notes

- Data is example-only (not from a real person).
- Use in-app import to load and validate parsing behavior.
- Local scratch folders remain ignored: `Import Samples/` and `test_data/`.
